# ML Models Integration Guide

## Overview

Three trained ML models are now available in `backend/models/`:

### 1. Job Acceptance Classifier
- **Model**: `acceptance_xgboost.json` (XGBoost)
- **Schema**: `acceptance_schema.json`
- **Accuracy**: 77.8%
- **Use Case**: Predicts whether to accept, reject, or prioritize job offers
- **Output**: {0: reject, 1: accept, 2: prioritize}

### 2. Price Predictor
- **Model**: `price_logistic_regression.pkl` (Logistic Regression)
- **Scaler**: `price_scaler.pkl` (StandardScaler)
- **Schema**: `price_schema.json`
- **Accuracy**: 47.8%
- **Use Case**: Predicts optimal price per compute unit
- **Output**: Recommended price (float)

### 3. Success Probability Predictor
- **Model**: `success_xgboost.json` (XGBoost)
- **Schema**: `success_schema.json`
- **Accuracy**: 64.1% | Precision: 78.1%
- **Use Case**: Predicts likelihood of job completion success
- **Output**: Success probability (0-1)

## Usage in Your Code

### ResearchAgent (Pricing)
```python
from intelligence.predictor import Predictor

# The Predictor automatically loads the trained models
predictor = Predictor(
    model_path="backend/models/acceptance_xgboost.json",
    feature_schema_path="backend/models/acceptance_schema.json"
)

# When predicting, it will use the trained XGBoost model
result = predictor.predict(features)
```

### ComputeAgent (Risk Assessment)
```python
from intelligence.risk_agent import RiskAgent

risk_agent = RiskAgent()
decision = risk_agent.evaluate_job(
    job_spec=job,
    agent_state=agent_state
)
# Uses success predictor internally
```

## Retraining Models

To retrain with new data:

```bash
cd backend
python train_models.py
```

This will:
- Generate 5000 synthetic training samples
- Train all three models
- Save to `backend/models/`
- Generate `training_summary.json`
